## sh脚本

点击一键自动提交，后续可添加gitwatch监视修改自动提交

```
#!/bin/bash

cd D:/study/hexoBlog  # 切换目录

if [ -z $(git status -s) ]; then  # 如果没有改变，就不提交
  echo "No changes detected, skipping commit and push."
  exit 0
fi

git add .  # git提交命令

git commit -m "auto"

git push

```
